# Bhavya Fashion Design Institute (Nepal)

Official website for **Bhavya Fashion Boutique and Training Center** — fashion design training, tailoring programs, and boutique services in बुटवल 9 कालिकानगर, Nepal.

## Tech stack

- React (Vite)
- TypeScript
- Tailwind CSS
- React Router

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy

Configured for [Vercel](https://vercel.com). Connect this repository and deploy — Vite is auto-detected.
